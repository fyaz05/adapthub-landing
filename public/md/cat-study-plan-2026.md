# CAT 2026 study plan (free phase-wise)

**URL:** https://adapthub.in/cat-study-plan-2026/  
**Updated:** 2026-07-13

## Direct answer

A practical CAT 2026 study plan uses three phases: (1) foundation and diagnostic baseline, (2) adaptive intensification with weekly mocks, (3) mock-heavy refinement before the exam. Default length is about six months; compress only with a strong baseline. Every week should include adaptive practice, an error log, and scheduled mock analysis—not only hours of passive video.

## Weekly non-negotiables

- Adaptive drills on weak topics  
- Error log from mocks/sectionals  
- One analysis block after each full mock
