# AdaptHub Landing

High-performance marketing site for AdaptHub — free AI adaptive learning for CAT.

**Live:** [adapthub.in](https://adapthub.in) · **App:** [app.adapthub.in](https://app.adapthub.in)

## Core Methodology

- **Adaptive Difficulty**: Real-time calibration of challenge levels to maintain terminal flow state (ZPD optimization).
- **Dynamic Roadmaps**: Personalized learning trajectories that restructure based on performance signals.
- **Neural Guidance**: Context-aware AI explanations targeting specific logic and memory gaps.
- **Performance Analytics**: Granular tracking across accuracy, velocity, and concept mastery.
- **Mastery-First Design**: Engineered for high-density retrieval and long-term retention.

## Tech Stack

- **Framework**: [Astro 5](https://astro.build/) + React 19
- **Runtime**: [Bun](https://bun.sh/)
- **Styling**: Tailwind CSS 4
- **Linting/Formatting**: [Biome](https://biomejs.dev/)
- **Animations**: [Motion](https://motion.dev/)
- **Deployment**: [Cloudflare Pages](https://pages.cloudflare.com/)

## Getting Started

First, install dependencies:

```bash
bun install
```

Run the development server:

```bash
bun run dev
```

### Quality Control

Check for linting and formatting issues:

```bash
bun run lint
```

Automatically fix formatting:

```bash
bun run format
```

## Deployment

To build and deploy to Cloudflare Pages:

```bash
bun run pages:deploy
```

The production site is live at [adapthub.in](https://adapthub.in).
